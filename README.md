  ### 🎥🌟 meerkats: 영화 예고편과 함께 하는 영화 커뮤니티 🌟🎥
  <br />
  <p>meerkats는 영화 예고편을 감상하고, 다양한 영화 정보와 소식을 공유하는 커뮤니티 플랫폼입니다.</p>
  <p>코로나19 팬데믹과 OTT 의 등장으로 인해 영화 산업이 변화하고 있지만, 여전히 영화관에서의 ‘몰입’과 ‘체험’을 소중히 여기는 사람들이 있습니다.</p>
  <p>meerkats 는 영화를 사랑하는 사람들에게 기다리던 영화가 개봉한다는 소식과 예고편을 가장 먼저 공개하고, 자유롭게 의견을 나눌 수 있는 소통의 장을 제공합니다.</p>
  <p>영화 예고편 및 관련 영상을 모아 볼 수 있는 기능과 함께, 오프라인에서 경험할 수 있는 다양한 이벤트와 콘텐츠를 소개합니다.</p>
  <p>meerkats 블로그는 자신의 취향과 감상을 공유하는 곳입니다. 유저는 영화에 대한 뜨거운 토론과 감상평을 나누며 커뮤니티를 만들어 갑니다.</p>
  <p>meerkats 와 함께 영화를 즐겨보세요!</p>
  </div>

  <h1 align="left">🖥️ Pages</h1>
  <h3><a href='https://github.com/mgkkm/meerkats-frontend'>meerkats 공통 레포가기</a>
  <h2>내가 구현한 페이지 &기능 </h2>
  <br />
  <h2>⓿ 헤더 </h2>
  <p>- 유저의 편의성을 높이기 위해 스크롤 방향에 따라 헤더를 보이거나 숨길 수 있도록 구현했습니다.</p>
  <p>- deltaY 속성 사용해서 scroll 상태 감지할 수 있는 기능 구현 완료
</p>
<p>
- 스크롤 감지 후 display 속성은 transition 을 적용할 수 없기 때문에 opacity속성 사용해  transition 효과 추가 완료
</p>
    <br />
  <h2>❶ 메인</h2>
  <p>- 가장 기대되는 개봉 예정작을 Swiper 라이브러리를 사용해 메인 캐러셀로 구현했습니다. 베스트 영화 또한 Swiper 라이브러리를 사용해 포스터가 순차적으로 보일 수 있도록 구현했습니다.</p>
  <p>
  - swiper의 loop 옵션을 사용하여 마지막 슬라이드가 끝나더라도 자연스럽게 첫 번째 슬라이드가 보여지도록 설정했고, autoplay 옵션을 사용해서 자동으로 슬라이드 넘어갈 수 있도록 했습니다.
  </p>
  <p>
  - 탭 기능을 사용해 각 카테고리 별로 인기 있는 영화들을 나열하고, 유저가 편하게 보고 싶은 영화를 고를 수 있도록 구현했습니다.
  </p>
  <p>
  - 모든 영화 포스터에 마우스를 올리면 각 영화의 상세페이지로 이동할 수 있도록 경로를 지정했습니다.
  </p>
   <br />  
  <h2>❷ 블로그 글 등록</h2>
  <p>- 글쓰기 창을 구현하기 위해 CK Editor 라이브러리를 도입하여, 블로그 글을 간편하게 작성할 수 있도록 했습니다.</p>
  <p>- 글 등록 및 수정 상태는 state로 관리했고, 상태에 따라 다양한 기능이 실행될 수 있도록 구현했습니다.</p>
  <p>- 좋은 사용자 경험을 제공하기 위해, 뒤로 가기 버튼 클릭 시 특정 페이지로 이동하는 대신 즉시 이전 페이지로 돌아갈 수 있도록 구현했습니다.</p>
 <p>- 글 수정 페이지에 유저가 글 등록 시 작성한 내용과 선택했던 카테고리 그대로 표시하여 편리하게 글을 수정할 수 있도록 했습니다.</p>
    <br />  
  <h2>❸ 이벤트 리스트</h2>
  <p>- 유저가 선택한 이벤트의 id 값을 받아 해당 디테일 페이지로 이동할 수 있도록 했습니다.</p>
  <p>- useNavigate / useParams 사용해서 페이지 이동 기능 구현 완료</p>
 <p>- HTMLButtonElement 타입에서 그냥 target 속성은 사용할 수 없기때문에 currentTarget을 사용하여 eventId를 가져옴.</p>
    <br />  
  <h2>❹ 이벤트 디테일</h2>
  <p>- 리스트 페이지에서 넘겨준 id 값을 받아 해당 영화에 대한 정보를 보여줄 수 있도록 구현했습니다.</p>
  <p>
  - react-youtube 라이브러리를 사용해 구현한 컴포넌트를 import 하여 영화 정보뿐만 아니라 영화의 예고편을 함께 보여주면서 이벤트와 영화에 대한 관심도를 높일 수 있도록 했습니다.
  </p>
    <br />  
<h2>❺ useAxios hook</h2>
<p>- 프론트에서 공통적으로 useAxios를 사용하기위해 useAxios hook을 만들어 사용했습니다.</p>
<p>- loading과 error 상태를 관리하기 위해 state를 사용했고, axios 요청의 상태를 추적할 수 있도록 구현하였습니다. </p>
<p>- 데이터를 관리하기 위해 data라는 state를 생성하고, axios를 통해 받아온 값이 setData를 통해 data state에 저장될 수 있도록 구현하였습니다. </p>
<p>- 이를 위해 fetchData라는 함수를 작성하여 axios를 실행할 수 있는 함수를 생성하였습니다.</p>
<br />
<h2>❻ comment component</h2>
<p>- 여러 페이지에서 공통적으로 사용되는 댓글 컴포넌트를 구현했습니다. 이를 통해 일관된 디자인과 사용자 경험을 제공할 수 있습니다.</p>
<p>- 블로그 댓글 등록 기능</p>
<p> 사용자가 입력한 댓글 내용을 서버로 전송하여 등록할 수 있도록 content state를 생성해서 관리했고, Recoil을 활용하여 서버로부터 받아온 블로그 댓글 정보를 공유하고 관리할 수 있도록 구현했습니다.</p>
<p>- 블로그 댓글 삭제 기능</p>
<p>삭제 버튼을 클릭하면 모달 창을 띄워 정말 삭제할건지에 대한 확인을 받아 댓글이 바로 삭제되는걸 방지했습니다. </p>
<p>선택한 댓글의 값을 e.target.value로 받아와 변수에 저장한 후, 서버에 삭제 요청을 보내기 위해 params를 사용했습니다.</p>
<p>- 블로그 댓글 수정 기능</p>
<p>Recoil을 사용하여 content state와 isEdit state를 생성하고, 수정 버튼을 누르면 기존 댓글 내용이 input창에 표시되어 바로 수정이 가능하도록 구현했습니다. </p>
<br />
<h2>modal</h2>
<p>- sweet alert 라이브러리를 이용해 반복적으로 사용되는 모달창을 공통함수로 만들었습니다.</p>
<p>- 사용자는 일관된 디자인의 모달창을 접해볼 수 있습니다.</p>
<p>- 함수의 인자로 title / text / naviagate / fetchData 함수를 props로 받아 여러 컴포넌트에서 확장성을 갖고 사용할 수 있게 구현했습니다.</p>
<br />
<h2>탑버튼</h2>
<p>- 사용자가 브라우저의 위치 700 이상으로 내려 온 경우에만 탑버튼이 보일 수 있도록 구현했습니다.</p>
<p>- window 객체의 scrollTo 옵션을 사용하여 탑버튼 클릭시 top: 0의 위치로 smooth하게 올라갈수 있도록 기능 구현 완료했습니다.</p>
